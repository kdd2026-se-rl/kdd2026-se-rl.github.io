Abstract

Automated RL Research via LLMs

We propose SE-RL, a framework that uses LLMs to automatically design, evaluate, and iteratively improve complete RL algorithms—including reward functions, agent architectures, and communication protocols—for optimal trade execution. Combined with a dynamic multi-agent market simulator and a hybrid training strategy, SE-RL outperforms all baselines on 200 stocks across CSI100 and NASDAQ100.

In quantitative finance, reinforcement learning (RL) has shown great promise for order execution due to its ability to interact with market environments based on real data. However, traditional RL methods suffer from slow research speed and rely on static market assumptions that do not consider the impact of an agent's execution actions on the environment.

To address these challenges, we propose a Self-Evolutional single-agent/multi-agent Reinforcement Learning (SE-RL) framework. The framework utilizes a Large Language Model (LLM) to design various RL algorithm modules—including agent model design, reward function, profiling, communication, and state imagination—by leveraging the LLM to generate module outputs or executable code.

SE-RL continuously improves the accuracy of LLM-generated RL algorithms through a Dual-Level Enhancement Kit (DEK) at both high-level (prompt refinement) and low-level (parameter fine-tuning). Additionally, we develop a dynamic financial market simulator based on a multi-agent system and introduce a hybrid environment training method that rebalances each environment's loss weight. Comprehensive experiments on 200 realistic stock datasets demonstrate that our proposed framework outperforms current state-of-the-art baselines.

Method

The SE-RL Framework

SE-RL is built upon a nested loop framework designed to iteratively enhance an LLM's capability as an automated RL algorithm designer. The outer loop enhances the LLM's research capabilities through the Dual-Level Enhancement Kit, while the inner loop trains an execution agent using the LLM-designed algorithm.

Figure 2 — Overview of the SE-RL Framework. The framework consists of two core schemes: (1) an outer loop for enhancing the LLM's research capabilities, and (2) an inner loop for training an execution agent with hybrid environment training.

Case Study

The Evolution of a Reward Function

SE-RL functions as a genuine "AI Researcher," automatically discovering core principles of quantitative trading through iterative evolution. Below we show how the reward function evolved from simple profit maximization to a sophisticated, volatility-aware risk management strategy.

Iteration 0 — Simple Profit

reward_v0.py 
def calculate_reward(self, pv_t, pv_t1):
    # Simple profit delta
    return pv_t - pv_t1

Iteration 8 — Evolved Strategy

reward_v8.py 
def calculate_reward(self, profit, inv, vol):
    reward = profit
    # Transaction cost awareness
    reward -= 0.001 * abs(profit)
    # Inventory risk penalty
    inv_pen = 0.05 * (inv ** 2)
    reward -= inv_pen
    # Volatility-adaptive scaling
    reward -= 0.1 * inv_pen * vol
    return reward
